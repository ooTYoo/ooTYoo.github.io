Github page + Hexo搭建个人博客

发表于 2018-07-01 更新于 2023-03-18 分类于 setup

简介

一直想搭个简单的个人博客，尝试过VPS等方式，最后选择Github Page+Hexo。该方式优点在于：稳定、免费、文档编辑部署都很简单。该组合可以满足个人博客需要的功能常见功能如:

  • 个人主页
  • 文章存档
  • 相关连接
  • 站内搜索、评论

下面对该搭建方式所需的基本要素进行简单介绍。并记录使用过程中踩过的坑，方便日后环境崩溃了进行修复。

组件简介

Git Pager
GitHub Pages本用于介绍托管在GitHub的项目，不过由于其免费稳定，适合用来做搭建博客。Github Pages可以被认为是用户编写的、托管在github上的静态网页。

NodeJs
Nodejs框架基于V8引擎，是目前速度最快的Javascript引擎（如Chrome浏览器就基于V8，可以流畅地同时打开数十个网页）。
Nodejs相比与标准的web开发框架Express，可以迅速建立web站点，且比起PHP学习曲线更低、开发效率更高。因此Nodejs非常适合小型网站，和个人主页网站。

Hexo
Hexo是一款基于Node.js的博客框架，其优势在于快速、简洁、高效。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内即可通过模版主题生成静态网页。

对于一些常用的模块，可以在Hexo中找到相关插件，作为对个人主页的补充。

更换主题

默认到Hexo主题为Landscape，主题相关配置文件存放在./themes路径下。界面美话相关的事情是最复杂的，而且很难用技术手段解决。幸运的是，诸多模版可以简化这个问题。你可以点这里浏览Hexo主题。

  • 选择喜欢的主题，down到./themes路径下，如Next。
  • 修改主资源目录下的_config.yml文件，将其中theme字段修改为你想用的主题。
  • 修改主资源目录下的_config.yml文件，添加对个人主页的一些说明。
  • 修改./themes/_config.yml以及相关资源文件，对主题进行调整。

常用的修改及关键字对应如下

  1. 更改语言为中文， language: zh-CN，支持的语言包位于/themes/next/languages下。
  2. 添加菜单按钮（如标签页、分类页等），在next主题下配置文件中，找menu下进行配置
  3. 产生标签页， hexo new page tags, 在tags/index.md头中添加type:"tags"
  4. 设置主题scheme: Gemini
  5. 添加头像Sidebar Avatar
  6. 设置打赏 reward
  7. 设置订阅 rss
  8. 背景设置 canvas
