RSA共模攻击

发表于 2021-09-10

在RSA公钥体系中，如果多个用户使用相同的模N作为其公钥中的模数，则存在共模攻击（Share Module Attack）。起原理为其中一个用户A可以使用自己的公私钥对 (eA,dA,N)(e_A, d_A, N)(eA​,dA​,N) 对 模数N 进行因式分解，求解得到 p，q 满足 N=p∗qN=p*qN=p∗q ，进一步使用该分解从 另一个用户B 的公钥 eBe_BeB​ 计算得到其对应的私钥dBd_BdB​。

对于同一个N，事实上相当于只有一对公私钥对可安全地用于RSA计算。

本文介绍共模攻击的原理和实施算法。

RSA算法基础

发表于 2021-03-18 更新于 2021-11-22

简介

RSA算法由Ron Rivest、Adi Shamir和Leonard Adleman于1977年提出，算法名字由三人名字开头字母组成。该算法是当前公钥密码体系中的算法基础。RSA算法的安全性依赖于“大整数因字分解问题是NP-hard的问题”这一假设。本文简单介绍RSA算法的数学基础，以及相关的一些算法知识。

发表于 2020-01-30 更新于 2022-03-17

gdb调试介绍

gdb(GNU Project debugger)，使得分析人员可以进入程序执行内部，暂停执行，查看和修改中间状态，通常用于软件调试问题定位。在GDB官网中介绍gdb可以完成的4种主要工作有：

  • 设置程序执行的上下文环境，并执行被调试程序
  • 让程序在特定条件下暂停，如执行到特定地址，满足特定条件等
  • 程序暂停时检查内部状态，如寄存器值，内存堆栈信息等
  • 修改程序内部状态，并继续执行
