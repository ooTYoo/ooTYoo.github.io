1. 原理介绍

Pohlig-Hellman 攻击，又称为sub-group攻击，是一种用来在有限域中求解离散对数的算法。

形式化得可以将该算法要解决的问题描述为，在有限群G中，对于给定的 g,h∈Gg,h\in Gg,h∈G，求解 a<ord(G)a<ord(G)a<ord(G) ，使得 ga=hg^a = hga=h，其中指数代表群G中a个g的群乘法操作。

根据定义，该算法可以用用在公钥系统如RSA、ECC等加密算法中。该算法的核心思想在于，如果 群G的阶数（G中元数目） n=ord(G)n = ord(G)n=ord(G) 可以进行如下因式分解，

baby-step giant-step 算法：

  • 输入：群G，阶数为n，群G的元素a,b
  • 输出：x， 使得 $a^x = b $
    1. 计算 m = ceiling (n\sqrt{n}n​)
    2. 预计算表[(j, a^j) for j in range(m)]
    3. 计算 A = a−ma^{-m}a−m
    4. 令R = b
    5. for i in range(m):
      1. check R == a^{j}
      2. 如果相等，return x = i*m + j
      3. 如果不等，R = b* A

4. 总结

Pohlig-hellman攻击说明，在公钥系统中，选择群的阶数对算法安全性有较大的影响。在RSA中通常N为两个素数的乘积，在一些情况下，安全性会受到该分解算法的影响。在ECC系统中，选取基点G是，应当进来保证G的阶尽量大，且尽量为素数，以此来提升系统的安全性。
