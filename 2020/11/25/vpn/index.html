vpn搭建梯子教程

发表于 2020-11-25 更新于 2022-02-19 分类于 setup

很多时候需要梯子去查找外部的资料。当然了科学上网的前提是要遵循基本的法律要求，这也是专业性的一种体现。

做一件事通常有多条路径。我们这里参考的是使用Trojan实现梯子，这种方式的好处是起有效性，通过将流量路由到自己的网站，避免了很多麻烦。

这里简单介绍下基本的步骤，仅供自己遇到问题时进行定位。

服务器配置

  1. 需要购买VPS服务器作为网络流量的出口，这里用的是vultr。域选择Jap，选择合适的套餐（5$/m的即可，1000G流量）
    • 位置选择Jap，速度会好些
    • OS选择CentOS 7x64
    • 套餐选择$5/m，该套餐最便宜，且1000GB带宽是够用的。
    • 需要的话，可以生成ssh-key，将公钥上传到服务端，好处是控制起来很方便。
    • 完成选择后，的deploy即可，需要花费3min左右时间。
  2. 需要有一个域名来搭建搭建网站，这个当前是个伪站，应该也可以做成个人主站。这里从腾讯云购买域名和DNS解析服务。
  3. 在腾讯云中，配置域名DNS的解析地址为vps的ip地址, 一般需要实名认证，配置后2～3天生效。
  4. vps上安装trojan脚本，该脚本会生成伪站，并生成对应的证书。

注意：如果系统时间不对，可能会导致网络无法使用。

  1. 过程中会提示需要输入域名，输入解析到本VPS的域名
  2. 完成后，会生成域名下的一个下载链接，下载得到trojan-cil压缩包
  3. 打开压缩包，找到config.json，记下remote_addr, remote_port, password几项。
  4. 注意脚本给自己的主站申请了证书，压缩包中的password使用随机数进行md5计算得到。
  5. 安装Ber网页加速器, 之后显示BBR启动成功，服务端架设的步骤就完成了。
    a. 安装Ber加速器时，需要修改内核模块 ，安装指导输入1时失败，导致服务器崩溃，可以先不启用。

linux客户端配置

在Linux客户端需进行配置，以便可以在terminal中使用代理。

1.下载编译好的(版本)[https://github.com/trojan-gfw/trojan/releases]，下载完成后解压。
2. 使用新的config.json文件替换其中内容，并且替换证书链fullchain.cer
3. cp trojan /usr/src/trojan
4. sudo ./trojan
5. 退出的方式：pkill -f trojan
6. 检查是否成功 ps aux | grep trojan | grep -v grep
7. 检查是否联通 curl ip.sb --socks5 127.0.0.1:1080

需要在terminal中使用代理

  1. 下载: sudo apt-get install proxychains
  2. 配置: sudo vi /etc/proxychains.conf
  3. 在最后的ProxyList里注释默认的socks代理： socks4 前增加#表示注释
  4. 在最后的ProxyList里加入Trojan的代理设置： socks5 127.0.0.1 1080
  5. 测试本地IP: curl -4 ip.sb，将显示本地IP
  6. 测试代理IP: proxychains curl -4 ip.sb，将显示Trojan代理的IP
  7. 后续使用的命令行需要代理时，只需要在前面加上 proxychains 即可如 proxychains npm install xxx

windows客户端

客户端使用v2rayN-with-Trojan for window

  • 双击打开trojan文件夹下的 v2rayN.exe 文件

  • 如客户端运行Trojan报错，请尝试下载以下软件，安装完毕再运行Trojan

    • Windows.NET Framework 4.6.2
    • visual C++ 2015 (x64、x86均安装)

  • 桌面右下角会出现1个小图标，再次双击，打开 Trojan 设置面板

  • 点击「服务器」-「Add [Trojan] server」来添加Trojan节点, 输入trojan-cli包中配置文件的addr，port，password。

  • 在服务器一栏勾选上已经配好的服务器，点击启用Http代理，代理模式选pac模式。之后就可以开心的科学上网了。

  • 不用的时候，按顺序先把启用http代理勾选取消，之后退出客户端，最后关电脑。

  • 如果有没有按顺序操作，再次开机发现上不了网了，请在这里的小电脑点右键，打开“网络和Internet设置”→代理→手动设置代理，将使用代理服务器一栏关闭
