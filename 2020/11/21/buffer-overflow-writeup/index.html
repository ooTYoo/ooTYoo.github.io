基本介绍

练习使用的镜像来自exploit-exercises中的Protostar，该镜像覆盖的练习内容包括

  • buffer overflows
  • format strings
  • heap exploitation
  • socket programming

镜像为linux-32bit系统，用户名密码为：user/user, root用户对应的密码为godmode。使用virtualbox安装镜像时，增添虚拟网卡，配置网络环境为Host-only Adapter。对应的可执行文件位于/opt/protostar/bin/下。

stack5

该题中无目标函数，需要攻击者将自己希望之星的代码通过输入放入stack中，并覆盖stack中存储的ip进行执行。这是典型的BoF攻击场景。题目中提到可以使用’\xcc’(int3)中段来调试恶意代码。

典型的BoF传入恶意代码的目的为获取shell，因此恶意代码又被称为shellcode。shellcode可以自己编写，也可以在网上直接查找，如 shellstorm，在上面可以找到对应linux/x86 sh相关的shellcode为

这个shellcode长度为21Byte，可以放入buffer中。接下来需要做的就是讲返回地址覆盖为buffer的地址即可。
